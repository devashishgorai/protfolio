This repository contains my personal developer portfolio, showcasing my projects, skills, achievements, and journey as an aspiring Software Engineer.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
